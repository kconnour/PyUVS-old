About PyUVS
===========

PyUVS is the brainchild of kconnour and zachariahmilby. It aims to be the
ultimate package for working with MAVEN/IUVS data, providing users standard
ancillary arrays, an interface that reduces the barrier for entry for working
with IUVS data, and a collection of some of IUVS' best graphics.
