constants¶

Collection of constants and conversion factors relevant to MAVEN/IUVS.

Most are specific to the IUVS instrument, but some help with general photometric calculations.

Instrumental constants¶

pyuvs.constants.angular_slit_width: float = 10.64¶

Width of the slit [degrees].

pyuvs.constants.cmos_pixel_well_depth: int = 3400¶

Saturation level of an IUVS CMOS detector pixel [DN].

pyuvs.constants.telescope_focal_length: int = 100¶

Focal length of the IUVS telescope mirror [mm].

pyuvs.constants.pixel_omega: float = 2.3438e-07¶

Detector pixel angular dispersion along the slit.

pyuvs.constants.pixel_size: float = 0.023438¶

Size of an IUVS detector pixel [mm].

pyuvs.constants.spatial_slit_width: float = 0.1¶

Width of the slit [mm].

Martian constants¶

pyuvs.constants.mars_mass: float = 6.417099999999999e+23¶

The mass of Mars [kg]. This value come from the Mars fact sheet.

pyuvs.constants.mars_mean_radius: float = 3389.5¶

The mean radius of Mars [km]. This value come from the Mars fact sheet.

Physical constants¶

pyuvs.constants.kR: float = 79577471.54594767¶

Definition of the kilorayleigh [photons/steradian].
