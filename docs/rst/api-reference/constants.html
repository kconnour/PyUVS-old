Constants¶

Collection of constants relevant to MAVEN/IUVS.

Most are specific to the IUVS instrument, but some help with general photometric calculations.

Instrumental constants¶

constants.angular_slit_width: float = 10.64¶

Width of the slit [degrees].

constants.cmos_pixel_well_depth: int = 3400¶

Saturation level of an IUVS CMOS detector [DN].

constants.focal_length: float = 100.0¶

Focal length of the IUVS telescope mirror [mm].

constants.pixel_omega: float = 2.3438e-07¶

Detector pixel angular dispersion along the slit.

constants.pixel_size: float = 0.023438¶

Size of an IUVS detector pixel [mm].

constants.spatial_slit_width: float = 0.1¶

Width of the slit [mm].

Physical constants¶

constants.kR: float = 79577471.54594767¶

Definition of the kilorayleigh [photons/steradian].

Martian constants¶

These values come from the Mars fact sheet.

constants.mars_mass: float = 6.417099999999999e+23¶

The mass of Mars [kg].

constants.mars_mean_radius: float = 3389.5¶

The mean radius of Mars [km].
