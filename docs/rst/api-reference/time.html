time¶

Collection of tools for converting between Earth and Martian times.

ScienceWeek¶

class pyuvs.time.ScienceWeek¶

Convert between dates and MAVEN science weeks.

This class contains a variety of methods that can either convert a science week number into an Earth date, or convert an Earth date into a MAVEN science weeks.

Notes

MAVEN’s science weeks initially ran Tuesday–Monday, but in June of 2021 they switched to a Thursday–Wednesday schedule.

property start_date: datetime.date¶

Get the date that MAVEN begin performing science.

week_from_date(date)¶

Compute the MAVEN science week number corresponding to an input date.

Parameters

date (datetime.date) – The date to get the science week from.

Raises

  • TypeError – Raised if date is not a datetime.date.

  • ValueError – Raised if date is before the start of IUVS science.

Return type

int

Warning

UserWarning

Raised if the input date was during the bridge phase.

Examples

>>> import datetime
>>> sw = ScienceWeek()
>>> sw.week_from_date(sw.start_date)
0
>>> sw.week_from_date(datetime.date(2020, 1, 1))
268
