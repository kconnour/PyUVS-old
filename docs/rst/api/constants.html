Constants¶

This module contains constants and conversions factors relevant to IUVS.

pyuvs.constants.angular_slit_width: float = 10.64¶

Width of the slit [degrees].

pyuvs.constants.spatial_slit_width: float = 0.1¶

Width of the slit [mm].

pyuvs.constants.cmos_pixel_well_depth: int = 3400¶

Saturation level of an IUVS CMOS detector pixel [DN].

pyuvs.constants.day_night_voltage_boundary: int = 790¶

Voltage defining the boundary between dayside and nightside settings.

pyuvs.constants.minimum_mirror_angle: float = 30.2508544921875¶

Minimum angle [degrees] the scan mirror can be.

pyuvs.constants.maximum_mirror_angle: float = 59.6502685546875¶

Maximum angle [degrees] the scan mirror can be.

pyuvs.constants.telescope_focal_length: int = 100¶

Focal length of the IUVS telescope mirror [mm].

pyuvs.constants.pixel_size: float = 0.023438¶

Size of an IUVS detector pixel [mm].

pyuvs.constants.pixel_omega: float = 2.3438e-07¶

Detector pixel angular dispersion along the slit.

pyuvs.constants.muv_wavelength_width: float = 0.16367098¶

Docstring.

pyuvs.constants.kR: float = 795774715.4594767¶

Definition of the kilorayleigh [photons/second/m**2/steradian].
