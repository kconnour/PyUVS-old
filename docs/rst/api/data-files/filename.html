filename¶

This module provides objects for parsing filenames.

class pyuvs.data_files.filename.DataFilename(path: str)[source]¶

A data structure containing info from a single IUVS filename.

It ensures the input filename represents an IUVS filename and extracts all information related to the observation and processing pipeline from the input.
