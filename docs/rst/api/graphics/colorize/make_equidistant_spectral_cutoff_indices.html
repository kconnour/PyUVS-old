make_equidistant_spectral_cutoff_indices¶

pyuvs.graphics.make_equidistant_spectral_cutoff_indices(n_wavelengths: int) → tuple[int, int][source]¶

Make the cutoff indices so that the spectral dimension can be downsampled to 3 equally spaced color channels.

Parameters

n_wavelengths (int) – The number of wavelengths.

Returns

The blue-green and the green-red cutoff indices.

Return type

tuple[int, int]

Examples

Get the wavelength cutoffs for IUVS 19 spectral binning scheme.

>>> import pyuvs as pu
>>> pu.graphics.make_equidistant_spectral_cutoff_indices(19)
(6, 12)
