load_standard_fit_templates¶

pyuvs.load_standard_fit_templates() → numpy.ndarray[source]¶

Load the standard fit templates.

The standard fit templates are:

  1. NO nightglow

  2. CO Cameron bands

  3. Ultraviolet doublet

  4. Solar continuum.

These correspond to the first axis.

Returns

The fit templates.

Return type

np.ndarray

Notes

This array has shape (4, 1024).
