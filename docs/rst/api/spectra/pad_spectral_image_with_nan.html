Pad the spectral axis of an image with NaNs.

The spectral axis is assumed to be the last axis, consistent with the IUVS standard.

The input array with NaNs padded on the ends.

Notes

IUVS takes data with different spectral binning schemes. Frequently, it doesnât have the bandwidth to transmit all the spectral data, so it only transmits a fraction of that data. This function puts the image in the place of a hypothetical array where it transmitted all data.

This is particularly useful when fitting spectral templates to data.

Examples

Fit an image of shape (200, 50, 40) into a 256 spectral bin array that transmitted data starting at the 60th spectral bin.

>>> import numpy as np
>>> import pyuvs as pu
>>> dummy_data = np.ones((200, 50, 40))
>>> padded_data = pu.pad_spectral_image_with_nan(dummy_data, 256, 60)
>>> padded_data.shape
(200, 50, 256)
>>> padded_data[0, 0, 59], padded_data[0, 0, 60]
(nan, 1.0)
