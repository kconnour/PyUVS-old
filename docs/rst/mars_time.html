Mars Time¶

The time module contains classes for converting Earth times into Martian times.

class mars_time.ScienceWeek¶

ScienceWeek can convert between dates and MAVEN science weeks.

ScienceWeek contains a variety of methods that can either convert a week number into a datetime.date object representing the corresponding Earth date, or convert a datetime.date object into MAVEN science weeks.
